TASKFORCE CHALLENGE FRONT-END

Description

This is an app which present the status of covid 19 pandemic around the world. You can get data of any continent, country and some cities on this site.

Installation

  • Clone this repository from your CLI by git clone https://github.com/mansurissa/taskforce-challenge-frontend
  • cd taskforce-challenge-frontend
  • Install dependencies using npm install command.

API

API that provide covid-19 data can be found here

Starting the application

  • Run npm start To start your application on localhost:3000.

Testing

  • Use npm run test to run the application's test.

Documentation

  • Use npm run docs to run get the app's documentaion.
